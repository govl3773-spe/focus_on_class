# 0_prototype

## 브랜치 목적

초기 MediaPipe 기반 프로토타입과 얼굴 랜드마크 분류 아이디어를 빠르게 검증하기 위한 실험 브랜치입니다.

## develop 포함 여부

이 브랜치는 초기 프로토타입 실험용 브랜치이므로 `develop` 브랜치에 포함하지 않습니다.

## 주요 실험 내용 요약

- `1-0_prototype.ipynb`: 초기 프로토타입 실험 노트북
- `1-1_us_proto.ipynb`: MediaPipe 기반 웹캠/전처리 실험 노트북
- `5-4_face_classirier.ipynb`: 얼굴 랜드마크 기반 분류기 실험 노트북
- `mp_data/`, `mp_model/`: 프로토타입 실험 중 생성된 데이터와 모델 자산
- `docs/`: 프로토타입 검토, 변경 방향, MediaPipe 관련 정리 문서
